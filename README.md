# Rodrigo Digital

## Descripción

Chatbot self-hosted con API Gemini.

## Características

- Manejo de API key local (no almacenada en el repositorio)
- Despliegue con GitHub Pages
- Interfaz estática HTML/JavaScript

## Getting Started

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/James-gosling/rodrigo-digital.git
   cd rodrigo-digital
   ```

2. Obtener una API key de [Google AI Studio](https://makersuite.google.com/app/apikey)

3. Pegar la API key en la aplicación cuando se solicite

4. Comenzar a chatear con el asistente
